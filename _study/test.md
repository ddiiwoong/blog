## What is ETCD
A distributed reliable, simple, secure, fast key-value store 
* Tabular/Relational Databases와 Key-value store 차이점 이해

## ETCD 설치

1. Download Binaries
2. Extract
3. Run ETCD Service
  ```sh
  $ /bin/etcd
  $ etcdctl set key1 value1
  $ etcdctl get key1
  value1
  $ etcdctl
  ```
